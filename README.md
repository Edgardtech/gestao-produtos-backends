## 📁 PROJECT 3 — AuthApp (Sistema com Login)
> Stack: Node.js + Express + SQLite + bcrypt + sessions

```bash
cd project3
npm install
node server.js
```
Acesse: http://localhost:3000

**Funcionalidades:**
- Cadastro de usuário (senha criptografada com bcrypt)
- Login com sessão
- Dashboard protegido
- Logout

---

## 📁 PROJECT 4 — Gestão de Produtos (CRUD)
> Stack: Node.js + Express + SQLite

```bash
cd project4
npm install
node server.js
```
Acesse: http://localhost:3001

**Funcionalidades:**
- Listar produtos com busca em tempo real
- Criar produto (modal)
- Editar produto (modal)
- Deletar produto (confirmação)
- Stats: total, em estoque, estoque baixo, sem estoque

---
